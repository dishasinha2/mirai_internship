# 🌌 Multiverse AI Chatbot

An interactive AI chatbot built with **Streamlit** and **Google Gemini API** that allows users to chat with different AI personalities in a modern chat interface.

> Developed as part of the **MirAI School of Technology – Virtual Summer Internship 2026**.

---

## 🚀 Features

- 🎭 Multiple AI Personalities
  - Virat Kohli
  - Anushka Sharma
  - Elvish Yadav
  - Pritam & Pedro
  - Albert Einstein
  - Iron Man

- 💬 ChatGPT-style Interface
- 🧠 Conversation Memory
- ⚡ Google Gemini 2.5 Flash Integration
- 📊 Token Usage Estimator
- 🎨 Beautiful Streamlit UI
- ⏳ Loading Spinner
- 📥 Download Chat History
- 🗑️ Clear Chat Button
- 🔒 Secure API Key using `.env`

## 📦 Installation

### Clone the repository

```bash
git clone https://github.com/dishasinha2/mirai_internship.git
```

### Move into the project directory

```bash
cd mirai_internship
```

### Install dependencies

```bash
pip install -r requirements.txt
```

---

## 🔑 Environment Variables

Create a `.env` file in the project root.

```env
GEMINI_API_KEY=YOUR_GEMINI_API_KEY
```

> **Never upload your `.env` file to GitHub.**

---

## ▶️ Run the Application

```bash
streamlit run app.py
```

After running, open the local URL shown in the terminal (usually `http://localhost:8501`).
